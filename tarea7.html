Explora los sensores, actuadores y hardware compatible con la plataforma Arduino para construir proyectos electrónicos increíbles.

La Placa Arduino

Base
🟦

Arduino UNO R3 — La más popular

Microcontrolador basado en el ATmega328P. Es la placa de referencia de Arduino, ideal para aprender electrónica y programación. Cuenta con 14 pines digitales, 6 entradas analógicas y comunicación USB integrada.

Procesador ATmega328P
Velocidad 16 MHz
Pines digitales 14
Entradas analógicas 6
Flash 32 KB
Voltaje 5 V
Principiantes USB-B Open Source
🟩

Arduino Nano

Versión compacta del UNO, ideal para proyectos donde el espacio es limitado. Mismo procesador pero en un factor de forma diminuto, perfecto para wearables y gadgets pequeños.

CompactoBreadboardATmega328
🟧

Arduino Mega 2560

Para proyectos complejos que necesitan más pines y memoria. Cuenta con 54 pines digitales y 16 analógicos, perfecto para impresoras 3D, CNC y robots avanzados.

54 pinesAvanzado256 KB Flash

Sensores

Entrada de datos
🌡️

Sensor de Temperatura DHT11 / DHT22

Mide temperatura y humedad relativa del ambiente. El DHT22 ofrece mayor precisión (±0.5°C). Se conecta a un pin digital y usa su propio protocolo de comunicación de 1 cable.

DigitalTemperaturaHumedad
📏

Sensor Ultrasónico HC-SR04

Mide distancias de 2 cm a 4 metros usando ondas ultrasónicas. Muy usado en robots para evitar obstáculos. Funciona enviando un pulso y midiendo el tiempo de regreso del eco.

Distancia2–400 cmTrigger/Echo
💡

Sensor de Luz LDR (Fotorresistor)

Resistencia variable según la intensidad de luz. Al combinarlo con un divisor de voltaje y una entrada analógica de Arduino, podemos leer el nivel de luminosidad del entorno.

AnalógicoLuminosidadBajo costo
🔴

Sensor de Movimiento PIR HC-SR501

Detecta movimiento de cuerpos con calor infrarrojo. Ideal para sistemas de alarma, luces automáticas y proyectos de seguridad. Salida digital: HIGH cuando detecta movimiento.

DigitalInfrarrojoAlarmas
🌱

Sensor de Humedad del Suelo

Mide el nivel de humedad de la tierra mediante resistividad eléctrica. Perfecto para sistemas de riego automatizado y huertos inteligentes. Salida analógica y digital.

JardineríaAnalógicoIoT
🔊

Sensor de Sonido KY-038

Detecta el nivel de ruido ambiental. Tiene potenciómetro para ajustar sensibilidad. Salida analógica para medir intensidad y salida digital para umbral de activación.

AudioDoble salidaAjustable
🧲

Sensor Hall (Magnético)

Detecta campos magnéticos. Se usa para contar revoluciones de motores, detectar la posición de imanes y construir encoders. Salida digital activa con presencia de imán.

MagnéticoRPMPosición
🌊

Sensor de Flujo de Agua YF-S201

Mide el caudal de agua en tuberías usando una rueda de paletas con sensor Hall interno. Muy utilizado en sistemas de monitoreo de consumo de agua y dispensadores.

AguaPulsos1–30 L/min

Actuadores

Salida / Acción
💦

Servomotor SG90 / MG996R

Motor con control preciso de posición angular (0°–180°). El SG90 es pequeño y ligero; el MG996R es metálico y más potente. Se controlan con señal PWM desde Arduino.

PWM0–180°Robótica
⚡

Motor DC + Driver L298N

Los motores de corriente directa requieren el módulo driver L298N para ser controlados desde Arduino. Permite controlar velocidad con PWM y dirección de giro de 2 motores simultáneamente.

DriverPWMVehículos
🔔

Buzzer Activo / Pasivo

El buzzer activo genera un tono fijo al aplicar voltaje. El pasivo necesita una señal PWM para producir diferentes frecuencias, permitiendo crear melodías y alertas sonoras.

SonidoAlertasDigital
🔄

Relevador (Relay) 5V

Permite a Arduino controlar dispositivos de alto voltaje (110/220V AC) de forma segura. Un pequeño voltaje de 5V controla un circuito de alta potencia, como focos o electrodomésticos.

Alto voltajeAislamientoAutomatización

Módulos de Comunicación

Conectividad
📶

Módulo WiFi ESP8266 / ESP-01

Agrega conectividad WiFi a Arduino. Permite enviar datos a internet, controlar el Arduino desde un celular o crear un servidor web local. Comunica vía Serial (AT commands).

WiFiIoTSerial
📡

Módulo Bluetooth HC-05 / HC-06

Permite comunicación inalámbrica a corto alcance (~10 m). HC-05 puede ser maestro o esclavo; HC-06 solo esclavo. Ideal para controlar Arduino desde una app en el celular.

Bluetooth 2.1SerialApp control
🛰️

Módulo GPS NEO-6M

Recibe señales de satélites GPS para obtener coordenadas de latitud y longitud, altitud y velocidad. Perfecto para rastreadores de vehículos y proyectos de geolocalización.

GPSUARTRastreo
📻

Módulo NRF24L01 (Radio 2.4 GHz)

Comunicación inalámbrica de bajo costo y consumo. Hasta 250 kbps de velocidad de datos y alcance de 100+ metros en campo abierto. Comunicación bidireccional vía SPI.

2.4 GHzSPI100m

Pantallas y Displays

Visualización
📺

LCD 16×2 con I2C

Pantalla de cristal líquido de 16 columnas y 2 filas. Con el módulo I2C solo requiere 2 cables de datos (SDA/SCL), ahorrando muchos pines digitales del Arduino. Muestra texto y caracteres.

I2CTextoBajo consumo
🖥️

Pantalla OLED 0.96" SSD1306

Pantalla OLED de alta resolución (128×64 px) con librería Adafruit SSD1306. Muestra texto, gráficos, iconos y animaciones. Comunica por I2C o SPI. Excelente contraste sin retroiluminación.

OLED128×64I2C/SPI
🔢

Display 7 Segmentos

Muestra dígitos del 0 al 9 mediante 7 LEDs en forma de segmentos. Se usa para visualizar contadores, temporizadores y mediciones numéricas simples. Versiones de 4 dígitos con chip TM1637.

TM1637Contadores2 cables
