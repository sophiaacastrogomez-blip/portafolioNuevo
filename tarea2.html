Un lenguaje de programación o diseño es un sistema formal de instrucciones que permite comunicarle órdenes a una computadora. Así como los humanos usamos idiomas para entendernos, los desarrolladores y diseñadores usan lenguajes para crear sitios web, aplicaciones, videojuegos y experiencias visuales. En esta tarea exploramos los más importantes agrupados en tres categorías: desarrollo web, aplicaciones de escritorio y diseño y maquetación.

🌐 Desarrollo Web

Los lenguajes que construyen todo lo que ves en un navegador

HTML
Lenguaje de marcado

HTML (HyperText Markup Language) es el esqueleto de toda página web. Define la estructura y el contenido: títulos, párrafos, imágenes, enlaces y formularios.

index.html
<!-- Mi primera página web --> <!DOCTYPE html> <html lang="es"> <head> <title>Cultura Digital</title> </head> <body> <h1>Hola Mundo 👋</h1> <p>Mi primera página web.</p> <a href="#">Clic aquí</a> </body> </html>
Páginas web Correos HTML Apps híbridas
CSS
Hoja de estilos

CSS (Cascading Style Sheets) le da colores, tipografías, animaciones y layouts al HTML. Es el "maquillaje" de la web: sin él, todo sería texto negro sin forma.

estilos.css
/* Estilo de un botón */ .boton { background: linear-gradient(#9b6bcc, #7a4aaa); color: white; padding: 10px 24px; border-radius: 30px; font-size: 1rem; transition: all 0.3s ease; } .boton:hover { transform: scale(1.05); }
Diseño web Animaciones Responsive
JavaScript
Lenguaje de scripting

JavaScript es el lenguaje que da interactividad y vida a las páginas. Con él funcionan los menús desplegables, validaciones de formularios, juegos y apps completas en el navegador.

app.js
// Función de saludo personalizado const saludar = (nombre) => { const hora = new Date().getHours(); let msg = hora < 12 ? "¡Buenos días" : "¡Buenas tardes"; return `${msg}, ${nombre}!`; }; console.log(saludar("Sophia"));
Frontend Backend (Node.js) Apps móviles
PHP
Lenguaje servidor

PHP es el lenguaje de servidor más usado en la web. Potencia WordPress, Facebook en sus inicios y millones de sitios. Procesa formularios, bases de datos y genera HTML dinámico.

index.php
<?php // Código del servidor $nombre = "Sophia Castro"; $materia = "Cultura Digital"; function presentar($n, $m) { return "Hola, soy {$n}" . " y estudio {$m}."; } echo presentar($nombre, $materia); ?>
WordPress Servidores web CMS
Python
Multiusos / Web / IA

Python es el lenguaje más popular del mundo actualmente. Se usa en web (Django, Flask), ciencia de datos, inteligencia artificial y automatización por su sintaxis simple y legible.

programa.py
# Lista de lenguajes favoritos lenguajes = ["HTML", "CSS", "Python"] def mostrar_lista(items): for i, lang in enumerate(items, 1): print(f"{i}. {lang}") mostrar_lista(lenguajes) # 1. HTML 2. CSS 3. Python
IA & ML Web (Django) Ciencia de datos
TypeScript
Superset de JS

TypeScript es JavaScript con tipado estático. Hace el código más robusto y predecible, ideal para proyectos grandes. Es el lenguaje principal de Angular y muy usado con React.

componente.ts
// Interfaz y función tipada interface Estudiante { nombre: string; edad: number; activo: boolean; } const registrar = (e: Estudiante): string => { return `${e.nombre}, ${e.edad} años`; }; registrar({ nombre: "Sophia", edad: 16, activo: true });
Angular React Proyectos grandes

🖥️ Desarrollo de Escritorio

Lenguajes para crear aplicaciones que se instalan en tu computadora

Java
Orientado a objetos

Java es el lenguaje de las grandes empresas. Con el lema "escribe una vez, ejecuta en cualquier parte", funciona en Windows, Mac, Linux y Android. Ideal para apps robustas.

Main.java
// Clase principal en Java public class Calculadora { private int resultado = 0; public int sumar(int a, int b) { resultado = a + b; return resultado; } public static void main(String[] args) { System.out.println("Suma: " + 5+3); } }
Android Servidores Apps empresariales
C Sharp
Ecosistema Microsoft

C# es el lenguaje de Microsoft para crear apps de Windows, videojuegos con Unity y aplicaciones empresariales. Combina la potencia de C++ con la facilidad de uso de Java.

Programa.cs
// Clase en C# using System; class Estudiante { public string Nombre { get; set; } public int Edad { get; set; } public void Presentar() { Console.WriteLine( $"Soy {Nombre}, tengo {Edad} años" ); } }
Unity Windows Apps .NET
Swift
Ecosistema Apple

Swift es el lenguaje de Apple para crear apps de iPhone, iPad y macOS. Es moderno, rápido y seguro, diseñado para reemplazar al antiguo Objective-C con una sintaxis mucho más amigable.

app.swift
// Estructura en Swift struct Tarea { var titulo: String var completada: Bool = false mutating func completar() { completada = true print("✅ \(titulo) lista!") } } var t = Tarea(titulo: "Cultura Digital") t.completar()
iOS macOS iPadOS
Kotlin
Android oficial

Kotlin es el lenguaje oficial de Android desde 2017. Más conciso que Java, elimina el código repetitivo y tiene soporte nativo para programación funcional y coroutines.

Main.kt
// Data class en Kotlin data class Usuario( val nombre: String, val edad: Int ) fun main() { val user = Usuario("Sophia", 16) val saludo = "Hola ${user.nombre}!" println(saludo) }

🎨 Diseño y Maquetación

Lenguajes y herramientas para crear la capa visual de lo digital

SVG
Gráficos vectoriales

SVG (Scalable Vector Graphics) es un lenguaje XML para gráficos vectoriales que se escalan perfectamente a cualquier tamaño sin perder calidad. Ideal para íconos, logos e ilustraciones web.

grafico.svg
<!-- Ilustración en SVG --> <svg viewBox="0 0 100 100" xmlns="http://www.w3.org/2000/svg"> <circle cx="50" cy="50" r="40" fill="#9b6bcc"/> <text x="50" y="55" text-anchor="middle" fill="white" font-size="14"> Hola SVG </text> </svg>
Íconos Animaciones Infografías
SASS
Preprocesador CSS

SASS/SCSS es un superlenguaje de CSS que añade variables, funciones, bucles y anidamiento. Hace que el código de estilos sea más organizado, reutilizable y fácil de mantener.

estilos.scss
// Variables y anidamiento SCSS $color-ppal: #9b6bcc; $radio: 12px; .tarjeta { border-radius: $radio; background: $color-ppal; &:hover { background: darken($color-ppal, 10%); transform: scale(1.02); } &__titulo { font-size: 1.2rem; } }
Diseño web avanzado Sistemas de diseño
Markdown
Texto enriquecido

Markdown es un lenguaje de marcado ligero que convierte texto plano en HTML formateado. Se usa en README de GitHub, blogs, documentación y plataformas como Notion o Discord.

README.md
# Título Principal (H1) ## Subtítulo (H2) Texto normal con **negrita** y *cursiva*. - Elemento de lista - Otro elemento - Sub-elemento [Enlace](https://ejemplo.com) ![Imagen](foto.jpg) `código inline`
Documentación GitHub Blogs
JSON / XML
Intercambio de datos

JSON y XML son formatos de datos estructurados. JSON es el estándar moderno para APIs y configuraciones; XML se usa en documentos, configs de apps y servicios SOAP legados.

datos.json
// Estructura JSON { "estudiante": { "nombre": "Sophia Castro", "grado": 2, "activa": true, "materias": [ "Cultura Digital", "Inteligencia Artificial" ] } }
