Comenzar
CSS3 (Cascading Style Sheets, nivel 3) es el lenguaje que le da estilo, color, tipografía, animaciones y estructura visual a las páginas HTML. Mientras HTML define qué hay en una página, CSS define cómo se ve. Sin CSS, la web sería texto negro sobre fondo blanco sin ningún diseño. En esta guía aprenderás todo desde cero: desde escribir tu primera regla hasta crear layouts complejos y animaciones profesionales.
0

¿Qué es CSS y cómo funciona?

Entiende el concepto antes de escribir una sola línea

CSS funciona con un principio de cascada: los estilos se aplican en orden de arriba a abajo, y los más específicos ganan sobre los generales. El nombre viene de ahí: Cascading Style Sheets.

La forma más profesional y recomendada es usar un archivo .css externo vinculado desde el HTML con <link>. Esto permite reutilizar los estilos en múltiples páginas y mantener el código organizado.

💡
La cascada y la especificidad: Cuando dos reglas CSS aplican al mismo elemento, gana la más específica. Orden de prioridad: !important → estilos inline → id → clases → etiquetas. Más sobre esto en el Paso 1.

Sintaxis CSS y Selectores

La gramática del lenguaje y cómo apuntar a los elementos HTML

selectores.css
/* ─── SELECTOR DE ETIQUETA ─── */ p { color: #333; } /* ─── SELECTOR DE CLASE (punto antes del nombre) ─── */ .boton-principal { background: #264de4; color: white; padding: 10px 24px; } /* ─── SELECTOR DE ID (# antes del nombre) ─── */ #menu-principal { display: flex; gap: 20px; } /* ─── SELECTOR COMBINADO ─── */ nav a { text-decoration: none; } /* a dentro de nav */ section > h2 { font-size: 1.5rem; } /* h2 hijo directo de section */ h1 + p { margin-top: 0; } /* p inmediatamente después de h1 */ input[required]{ border-color: red; } /* inputs con atributo required */ /* ─── ESPECIFICIDAD (quién gana) ─── */ p { color: black; } /* 0-0-1 */ .texto { color: blue; } /* 0-1-0 ← gana */ #parrafo { color: red; } /* 1-0-0 ← gana más */
✅
Buena práctica: Usa principalmente clases (.nombre) para estilos reutilizables. Evita #id para CSS porque su alta especificidad dificulta sobreescribir estilos. Reserva los id para JavaScript.

Colores y Fondos

Todas las formas de definir colores y propiedades de fondo en CSS3

colores-fondos.css
/* ─── FORMATOS DE COLOR ─── */ .caja { color: #264de4; /* hex */ color: rgb(38, 77, 228); /* rgb */ color: rgba(38,77,228, .8); /* rgba con opacidad */ color: hsl(228, 74%, 52%); /* hsl: tono, saturación, luminosidad */ background-color: transparent; } /* ─── FONDOS ─── */ .hero { /* Color sólido */ background-color: #f0f4ff; /* Gradiente lineal */ background: linear-gradient(135deg, #264de4, #9b6bcc); /* Gradiente radial */ background: radial-gradient(circle, #264de4 0%, transparent 70%); /* Imagen de fondo */ background-image: url("fondo.jpg"); background-size: cover; /* cover | contain | 100% */ background-position: center; background-repeat: no-repeat; background-attachment: fixed; /* efecto parallax */ /* Opacidad del elemento completo */ opacity: 0.9; }
color
Color del texto del elemento.
background-color
Color de fondo sólido.
background-image
Imagen o gradiente de fondo.
background-size
Tamaño del fondo: cover, contain, px.
linear-gradient()
Gradiente en línea recta con ángulo.
radial-gradient()
Gradiente circular o elíptico.
opacity
Transparencia del elemento (0–1).
rgba() / hsla()
Color con canal alpha (transparencia).

Tipografía en CSS

Controla fuentes, tamaños, espaciados y estilos de texto

tipografia.css
/* ─── IMPORTAR FUENTE DE GOOGLE FONTS (en HTML <head>) ─── */ /* <link href="https://fonts.googleapis.com/css2?family=Poppins" rel="stylesheet"> */ body { font-family: 'Poppins', sans-serif; /* fuente personalizada + fallback */ font-size: 16px; /* tamaño base */ line-height: 1.6; /* interlineado */ color: #1a1a1a; } h1 { font-size: clamp(1.8rem, 5vw, 4rem); /* fluido responsive */ font-weight: 700; letter-spacing: -0.02em; text-transform: none; /* uppercase | capitalize | lowercase */ } .parrafo { font-size: 1rem; /* 1rem = 16px por defecto */ text-align: justify; text-indent: 2em; /* sangría primera línea */ word-spacing: 4px; letter-spacing: 0.05em; text-decoration: none; /* underline | overline | line-through */ white-space: normal; /* nowrap | pre | pre-wrap */ } .truncar { white-space: nowrap; overflow: hidden; text-overflow: ellipsis; /* agrega "..." al truncar */ }

El Box Model (Modelo de Caja)

El concepto más importante de CSS: todo en la web es una caja

box-model.css
/* ─── RESET UNIVERSAL (siempre al inicio) ─── */ * { margin: 0; padding: 0; box-sizing: border-box; /* el width incluye padding y border */ } .tarjeta { /* DIMENSIONES */ width: 300px; max-width: 100%; /* nunca más ancho que su contenedor */ height: auto; /* PADDING (shorthand: arriba/abajo izq/der) */ padding: 24px 20px; /* MARGIN (auto = centrar) */ margin: 0 auto 32px; /* arriba der/izq abajo */ /* BORDER */ border: 2px solid #264de4; border-radius: 16px; /* SOMBRA */ box-shadow: 0 8px 32px rgba(38,77,228, .2); /* DESBORDAMIENTO */ overflow: hidden; /* visible | hidden | scroll | auto */ }
⚠️
Regla de oro: Siempre agrega * { box-sizing: border-box; } al inicio de tu CSS. Sin esto, el padding y el border se suman al width declarado y los cálculos de diseño se vuelven confusos.

Display y Posicionamiento

Cómo se comportan y ubican los elementos en la página

display-position.css
/* ─── DISPLAY ─── */ .bloque { display: block; } .linea { display: inline; } .mixto { display: inline-block; } .oculto { display: none; } /* elimina del flujo */ .invisible{ visibility: hidden; } /* oculta pero ocupa espacio */ /* ─── POSICIONAMIENTO ─── */ .contenedor { position: relative; } /* ancla para absolutos hijos */ .badge { position: absolute; /* fuera del flujo, relativo al padre relative */ top: 10px; right: 10px; z-index: 10; /* capas: más alto = encima */ } .navbar { position: fixed; /* siempre visible al hacer scroll */ top: 0; left: 0; width: 100%; z-index: 1000; } .sticky-header { position: sticky; /* se pega al hacer scroll hasta cierto punto */ top: 0; }

Flexbox — Layout en 1 dimensión

El sistema más usado para alinear y distribuir elementos en una fila o columna

flexbox.css
/* ─── CONTENEDOR FLEX ─── */ .contenedor { display: flex; /* Dirección del eje principal */ flex-direction: row; /* row | column | row-reverse */ /* Alineación en eje principal (horizontal en row) */ justify-content: space-between; /* flex-start | center | space-around | space-evenly */ /* Alineación en eje cruzado (vertical en row) */ align-items: center; /* flex-start | flex-end | stretch | baseline */ /* Espacio entre elementos */ gap: 16px; /* Permitir salto de línea */ flex-wrap: wrap; /* nowrap | wrap | wrap-reverse */ } /* ─── ELEMENTOS HIJOS ─── */ .item { flex-grow: 1; /* cuánto crece para llenar el espacio */ flex-shrink: 1; /* cuánto se encoge si hay poco espacio */ flex-basis: auto; /* tamaño base antes de grow/shrink */ /* shorthand: */ flex: 1; /* equivale a flex: 1 1 0 */ } .item-especial { align-self: flex-end; /* sobreescribe align-items solo para este */ order: -1; /* cambia el orden visual (no el HTML) */ }
🧠
Truco para memorizar: justify-content controla el eje principal (la dirección del flex). align-items controla el eje cruzado (perpendicular). Si flex-direction: row, justify es horizontal y align es vertical.
7

CSS Grid — Layout en 2 dimensiones

El sistema más poderoso para crear layouts complejos en filas Y columnas

grid.css
/* ─── CONTENEDOR GRID ─── */ .pagina { display: grid; grid-template-columns: repeat(3, 1fr); /* 3 columnas iguales */ grid-template-rows: auto 1fr auto; /* header, contenido, footer */ gap: 16px; min-height: 100vh; } /* ─── UNIDADES ESPECIALES ─── */ /* 1fr = 1 fracción del espacio disponible */ .galeria { display: grid; grid-template-columns: repeat(auto-fill, minmax(250px, 1fr)); /* auto-fill: crea tantas columnas como quepan de mínimo 250px */ gap: 20px; } /* ─── POSICIONAR ÍTEMS ─── */ .header { grid-column: 1 / 4; } /* ocupa todas las columnas */ .sidebar { grid-column: 1; grid-row: 2; } .main { grid-column: 2 / 4; } /* ocupa 2 columnas */ .footer { grid-column: 1 / -1; } /* hasta la última columna */ /* ─── TEMPLATE AREAS (más legible) ─── */ .layout { display: grid; grid-template-areas: "header header header" "sidebar main main " "footer footer footer"; } .header { grid-area: header; } .sidebar { grid-area: sidebar; } .main { grid-area: main; } .footer { grid-area: footer; }

Pseudoclases y Pseudoelementos

Estiliza estados e inserta contenido decorativo sin tocar el HTML

pseudo.css
/* ─── PSEUDOCLASES ─── */ .boton { background: #264de4; transition: all .3s ease; } .boton:hover { background: #1a38c0; transform: translateY(-2px); } .boton:active { transform: scale(.97); } .boton:focus { outline: 3px solid rgba(38,77,228,.4); } li:nth-child(odd) { background: #f0f4ff; } /* impares */ li:nth-child(even) { background: white; } /* pares */ p:not(.especial) { color: #333; } /* todos menos .especial */ /* ─── PSEUDOELEMENTOS ─── */ .boton::before { content: "✨ "; /* OBLIGATORIO: content para ::before/after */ } .separador::after { content: ""; display: block; width: 60px; height: 3px; background: #264de4; margin: 12px auto; } ::selection { background: #264de4; color: white; } /* color al seleccionar texto */ input::placeholder { color: #aaa; font-style: italic; }

Transiciones y Animaciones CSS3

Dale vida al diseño con movimiento fluido sin una línea de JavaScript

animaciones.css
/* ─── TRANSITION (cambio suave entre dos estados) ─── */ .tarjeta { transition: all .3s ease; /* propiedad duración timing-function delay */ transition: transform .4s ease-out, box-shadow .4s ease; } .tarjeta:hover { transform: translateY(-8px) scale(1.02); box-shadow: 0 20px 40px rgba(38,77,228,.3); } /* ─── @KEYFRAMES (animación continua) ─── */ @keyframes aparecer { from { opacity: 0; transform: translateY(30px); } to { opacity: 1; transform: translateY(0); } } @keyframes girar { 0% { transform: rotate(0deg); } 100% { transform: rotate(360deg); } } @keyframes pulsar { 0%, 100% { transform: scale(1); } 50% { transform: scale(1.08); } } .elemento { animation: aparecer .8s ease both; /* nombre duración timing fill-mode */ animation-delay: .2s; animation-iteration-count: infinite; /* o un número: 3 */ animation-direction: alternate; /* normal | reverse | alternate */ animation-play-state: running; /* paused con JS */ } /* ─── TRANSFORM (no anima por sí solo, úsalo con transition o @keyframes) ─── */ .forma { transform: translateX(50px) translateY(-20px) rotate(45deg) scale(1.5) skewX(10deg); }

Variables CSS y Diseño Responsive

Variables nativas para sistemas de diseño y @media queries para adaptarse a cualquier pantalla

variables-responsive.css
/* ═══════════════════════════════ VARIABLES CSS (Custom Properties) ═══════════════════════════════ */ :root { /* Paleta de colores */ --color-primario: #264de4; --color-secundario:#9b6bcc; --color-fondo: #f0f4ff; --color-texto: #1a2a4a; /* Tipografía */ --fuente-ppal: 'DM Sans', sans-serif; --tam-base: 16px; --tam-h1: clamp(1.8rem, 5vw, 4rem); /* Espaciado */ --espacio-sm: 8px; --espacio-md: 16px; --espacio-lg: 32px; /* Efectos */ --radio: 12px; --sombra: 0 8px 32px rgba(38,77,228, .15); } /* Usar variables */ .boton { background: var(--color-primario); font-family: var(--fuente-ppal); padding: var(--espacio-sm) var(--espacio-md); border-radius: var(--radio); box-shadow: var(--sombra); } /* ═══════════════════════ MEDIA QUERIES — Responsive ═══════════════════════ */ /* MOBILE FIRST: escribir para móvil primero, luego agrandar */ .grid { display: grid; grid-template-columns: 1fr; /* móvil: 1 columna */ gap: 16px; } /* Tablet: ≥ 768px */ @media (min-width: 768px) { .grid { grid-template-columns: repeat(2, 1fr); } } /* Desktop: ≥ 1024px */ @media (min-width: 1024px) { .grid { grid-template-columns: repeat(3, 1fr); } } /* Pantallas grandes: ≥ 1280px */ @media (min-width: 1280px) { .contenido { max-width: 1200px; margin: 0 auto; } } /* Modo oscuro automático */ @media (prefers-color-scheme: dark) { :root { --color-fondo: #0a1228; --color-texto: #e0e8ff; } } /* Imprimir */ @media print { .navbar, .sidebar { display: none; } }
