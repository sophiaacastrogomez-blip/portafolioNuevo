Conceptos Generales de Python

Fundamentos del lenguaje de programación más popular del mundo, con ejemplos prácticos.

🐍
Concepto 01

¿Qué es Python?

Python es un lenguaje de programación de alto nivel, interpretado y de propósito general. Fue creado por Guido van Rossum en 1991. Es famoso por su sintaxis limpia y legible, similar al inglés, lo que lo hace ideal para principiantes y expertos.

Ejemplo — Primer programa print("¡Hola, mundo!")
print("Bienvenido a Python")
Salida ¡Hola, mundo!
Bienvenido a Python
🏷 Lenguaje interpretado
📦
Concepto 02

Variables y Tipos de Datos

Una variable es un espacio en memoria donde guardamos información. Python detecta automáticamente el tipo de dato (tipado dinámico). Los tipos principales son: int, float, str, bool y list.

Ejemplo — Declaración de variables nombre = "Ana" # str
edad = 20 # int
altura = 1.65 # float
activo = True # bool
print(type(nombre)) # <class 'str'>
Salida <class 'str'>
🏷 Tipado dinámico
➕
Concepto 03

Operadores

Los operadores permiten realizar operaciones con variables y valores. Existen operadores aritméticos (+, -, *, /), de comparación (==, !=, >, <) y lógicos (and, or, not).

Ejemplo — Operadores aritméticos a = 10
b = 3
print(a + b) # 13 → suma
print(a ** b) # 1000 → potencia
print(a % b) # 1 → módulo (residuo)
Salida 13   1000   1
🏷 Aritmética y lógica
🔀
Concepto 04

Condicionales (if / elif / else)

Los condicionales permiten tomar decisiones en el código. Si una condición es verdadera, se ejecuta un bloque de código; si no, se pasa al siguiente. Python usa la indentación (sangría) para delimitar los bloques.

Ejemplo — Calificación escolar calificacion = 85

if calificacion >= 90:
    print("Excelente 🌟")
elif calificacion >= 70:
    print("Aprobado ✅")
else:
    print("Reprobado ❌")
Salida Aprobado ✅
🏷 Control de flujo
🔁
Concepto 05

Ciclos (for / while)

Los ciclos repiten un bloque de código varias veces. for se usa cuando se conoce el número de iteraciones (por ejemplo, recorrer una lista). while se usa cuando se repite mientras una condición sea verdadera.

Ejemplo — for y while # Con for
for i in range(1, 4):
    print(f"Vuelta {i}")

# Con while
n = 3
while n > 0:
    print(n); n -= 1
Salida (for) Vuelta 1   Vuelta 2   Vuelta 3
🏷 Iteración
⚙️
Concepto 06

Funciones

Una función es un bloque de código reutilizable que realiza una tarea específica. Se definen con def y pueden recibir parámetros y devolver un valor con return. Evitan repetir código.

Ejemplo — Función saludo def saludar(nombre, edad):
    return f"Hola {nombre}, tienes {edad} años"

mensaje = saludar("Luis", 21)
print(mensaje)
Salida Hola Luis, tienes 21 años
🏷 Reutilización de código
📋
Concepto 07

Listas

Una lista es una colección ordenada y modificable de elementos. Se crean con corchetes [ ] y pueden contener cualquier tipo de dato. Son una de las estructuras más usadas en Python por su flexibilidad.

Ejemplo — Operaciones con listas frutas = ["mango", "fresa", "papaya"]
frutas.append("guayaba") # agregar
print(frutas[0]) # mango
print(len(frutas)) # 4
frutas.remove("fresa") # eliminar
Salida mango   4
🏷 Estructura de datos
📖
Concepto 08

Diccionarios

Un diccionario almacena datos en pares clave: valor. Se crean con llaves { } y son ideales para representar objetos del mundo real, como los datos de una persona. Se accede a los valores usando la clave.

Ejemplo — Datos de un estudiante alumno = {
    "nombre": "Sofía",
    "carrera": "Ing. en Sistemas",
    "promedio": 9.2
}
print(alumno["nombre"])
Salida Sofía
🏷 Clave — Valor
